Detailed Description

The class used for creating various defenses. Implements the IDefenseResponse interface to listen to messages. Add this to an empty GameObject and give it a name to create a new Defense. Tweak the various variables in the Inspector tab in the Unity Editor to customize the defense. Should not be in a scene at start. Should be instatiated from player interraction.
